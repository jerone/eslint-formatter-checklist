# eslint-formatter-checklist

> An ESLint formatter with checklist style.

## Installation

```shell
npm install eslint-formatter-checklist --save-dev
```

## Usage

Run:

```shell
npx eslint --format checklist .
```

Or add script in your `package.json`:

```json
"scripts": {
  "lint": "eslint -f checklist ."
}
```
